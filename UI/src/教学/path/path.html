📚 第一章：什么是SVG Path？

SVG Path（路径）是SVG中最强大和灵活的元素，可以绘制任何形状。它使用一系列命令来定义如何绘制线条和曲线。

🎯 核心概念： Path就像用笔在纸上画画，你可以移动笔（不画线），画直线，画曲线，闭合图形等。

基本语法

<svg width="200" height="200"> <path d="M 10 10 L 100 100" stroke="black" fill="none"/> </svg>

🔧 第二章：Path命令大全

命令 全称 参数 作用
M x y MoveTo x, y坐标 移动到指定点（不画线）
L x y LineTo x, y坐标 画直线到指定点
H x Horizontal x坐标 画水平线
V y Vertical y坐标 画垂直线
C x1 y1, x2 y2, x y CurveTo 两个控制点+终点 三次贝塞尔曲线
Q x1 y1, x y Quadratic 一个控制点+终点 二次贝塞尔曲线
A rx ry rotation large-arc sweep x y Arc 椭圆参数+终点 椭圆弧
Z ClosePath 无 闭合路径
💡 提示： 大写字母表示绝对坐标，小写字母表示相对坐标（相对于当前位置）

🌟 第三章：基础示例

示例1：简单直线

<path d="M 20 20 L 180 130" stroke="#e74c3c" stroke-width="3" fill="none"/> 解释： M 20 20 - 移动到(20,20) L 180 130 - 画线到(180,130)

示例2：多条线段（折线）

<path d="M 20 120 L 60 40 L 100 80 L 140 20 L 180 100" stroke="#9b59b6" stroke-width="3" fill="none"/> 解释：连续的直线段形成折线

示例3：闭合图形（三角形）

<path d="M 100 20 L 170 120 L 30 120 Z" stroke="#e67e22" stroke-width="3" fill="#f39c12" fill-opacity="0.3"/> 解释： Z - 自动连接到起点，闭合图形

🎭 第四章：曲线的艺术

示例4：二次贝塞尔曲线（Q命令）

<path d="M 20 120 Q 100 20 180 120" stroke="#e91e63" stroke-width="3" fill="none"/> 解释： M 20 120 - 起点 Q 100 20 180 120 - 二次贝塞尔曲线 控制点(100,20) 终点(180,120)

示例5：三次贝塞尔曲线（C命令）

<path d="M 20 120 C 20 20, 180 20, 180 120" stroke="#673ab7" stroke-width="3" fill="none"/> 解释： C 20 20, 180 20, 180 120 控制点1 控制点2 终点 三次贝塞尔曲线更平滑

示例6：圆弧（A命令）

<!-- 小弧 --> <path d="M 50 100 A 50 30 0 0 1 150 100" stroke="#00bcd4" stroke-width="3"/> <!-- 大弧（虚线） --> <path d="M 50 100 A 50 30 0 1 0 150 100" stroke="#ff5722" stroke-dasharray="5,5"/> A rx ry rotation large-arc sweep x y 50 30 0(旋转) 0(小弧) 1(顺时针) 150 100

🚀 第五章：复杂图形

示例7：心形（修复版）

<path d="M 100 120 C 100 120, 70 100, 70 75 C 70 50, 85 40, 100 55 C 115 40, 130 50, 130 75 C 130 100, 100 120, 100 120 Z" fill="#e74c3c" stroke="#c0392b"/> 💖 心形路径 - 更宽更自然的心形

示例8：星形

<path d="M 100 10 L 110 40 L 140 40 L 120 60 L 130 90 L 100 75 L 70 90 L 80 60 L 60 40 L 90 40 Z" fill="#f1c40f" stroke="#f39c12"/> ⭐ 连续直线段组成五角星

示例9：波浪线

<path d="M 10 75 Q 30 25, 50 75 Q 70 125, 90 75 Q 110 25, 130 75 Q 150 125, 170 75 Q 190 25, 210 75" stroke="#3498db" stroke-width="4" fill="none"/> 🌊 连续二次贝塞尔曲线创建波浪

✨ 第六章：Path动画

示例10：连续路径绘制动画（A1→A2→A3→...）

<!-- 每个路径段都有独立的动画延迟 --> <path id="path1" d="M 30 50 Q 60 20 90 30" stroke="#e91e63" stroke-dasharray="100" stroke-dashoffset="100"> <animate attributeName="stroke-dashoffset" values="100;0" dur="1s" begin="0s"/> </path> <path id="path2" d="M 90 30 C 120 40, 140 60, 150 80" stroke="#9c27b0" stroke-dasharray="100" stroke-dashoffset="100"> <animate attributeName="stroke-dashoffset" values="100;0" dur="1s" begin="1s"/> </path> 💡 关键技巧：使用不同的begin延迟时间 实现连续绘制效果

示例11：高级连续动画（无限循环）

<style> .segment { stroke-dasharray: 50; stroke-dashoffset: 50; animation: drawSegment 0.8s ease-out forwards; } .segment:nth-child(2) { animation-delay: 0s; } .segment:nth-child(3) { animation-delay: 0.8s; } .segment:nth-child(4) { animation-delay: 1.6s; } .segment:nth-child(5) { animation-delay: 2.4s; } .segment:nth-child(6) { animation-delay: 3.2s; } .segment:nth-child(7) { animation-delay: 4.0s; } @keyframes drawSegment { to { stroke-dashoffset: 0; } } /* 整个动画组的循环 */ #pathNetwork { animation: resetPaths 6s infinite; } @keyframes resetPaths { 0%, 15% { transform: scale(1); opacity: 1; } 85%, 100% { transform: scale(1.05); opacity: 0.8; } } </style>

🎓 第七章：实用技巧

💡 Path优化技巧

  • 使用相对坐标（小写字母）减少代码量
  • 合理使用H、V命令简化水平垂直线
  • 复杂图形考虑分解为多个简单path
  • 使用path编辑工具（如Adobe Illustrator）生成复杂路径

示例12：相对坐标对比

<!-- 绝对坐标（大写）--> <path d="M 20 30 L 50 30 L 50 60 L 80 60 L 80 90"/> <!-- 相对坐标（小写）更简洁 --> <path d="M 120 30 l 30 0 l 0 30 l 30 0 l 0 30"/> <!-- 进一步简化 --> <path d="M 120 30 h 30 v 30 h 30 v 30"/>

🎯 第八章：学习总结

✅ 基础命令

  • M - 移动笔触
  • L - 画直线
  • H/V - 水平/垂直线
  • Z - 闭合路径

🎨 曲线命令

  • Q - 二次贝塞尔曲线
  • C - 三次贝塞尔曲线
  • A - 椭圆弧
  • S/T - 平滑曲线
