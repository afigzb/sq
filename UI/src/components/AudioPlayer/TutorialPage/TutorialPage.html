AudioPlayer 组件教程

音频播放器 Web 组件使用指南

AudioPlayer 是一个一点也不强大的音频播放器 Web 组件，他支持的功能很多， 例如支持自动播放、自定义播放列表、多种播放模式、浮动播放器界面等特性。 采用数据驱动分层式开发 但自动播放功能是有问题的，受限于浏览器的安全行为，自动播放功能实际上不好实现 我这里不过是在lit的updata里面循环开启播放的操作，直到浏览器判断用户有交互，允许AudioPlayer播放为止 本教程通过实际示例展示其主要用法。
