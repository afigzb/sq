🎵 音量设置演示

1. 默认音量（50%）
不设置 initial-volume 属性时，默认音量为 50%
<audio-player></audio-player>
2. 低音量（20%）
通过 initial-volume="20" 设置初始音量为 20%
<audio-player initial-volume="20"></audio-player>
3. 高音量（80%）
通过 initial-volume="80" 设置初始音量为 80%
<audio-player initial-volume="80"></audio-player>
4. 自动播放 + 自定义音量（60%）
结合自动播放和自定义音量的完整示例
<audio-player initial-track-id="track_2" auto-play initial-volume="60" initial-play-mode="LIST_LOOP"> </audio-player>
📝 使用说明
initial-volume 属性接受 0-100 的数值：
  • 0：静音
  • 1-30：低音量
  • 31-70：中等音量
  • 71-100：高音量

注意：现在音量设置会同时更新显示值和实际音频元素的音量，解决了之前只更新显示的问题。
