🚀 CodePreview 组件示例

遵循单一职责原则的代码预览组件 - 只负责将代码转化为预览效果

📝 基本使用示例

❌ 错误处理示例

💻 基本使用代码

// 创建预览组件
const preview = new CodePreview('#container', {
    width: '100%',
    height: '400px',
    errorDisplay: true
});

// 渲染代码
preview.render(codeString);

⚙️ 配置选项

const options = {
    width: '100%',          // 宽度
    height: '400px',        // 高度
    sandbox: 'allow-scripts', // iframe沙箱设置
    errorDisplay: true,     // 是否显示错误
    onError: (error) => {}, // 错误回调
    onLoad: () => {}        // 加载完成回调
};

🔧 组件方法

// 渲染代码
preview.render(code);

// 获取当前代码
const code = preview.getCode();

// 设置尺寸
preview.setSize('800px', '500px');

// 销毁组件
preview.destroy();

🚀 快速创建

// 静态方法快速创建
const preview = await CodePreview.create(
    '#container', 
    htmlCode, 
    { height: '300px' }
);
