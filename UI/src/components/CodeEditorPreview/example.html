🚀 代码编辑预览器

基于 Web Components 的现代化代码编辑和实时预览工具

✨ 主要功能特性

🎨 语法高亮

基于 Prism.js，支持多种编程语言和主题

⚡ 实时预览

编辑代码时实时更新预览，支持防抖优化

📝 在线编辑

双击即可编辑，支持 Tab 缩进和快捷键

📁 文件导入

支持外部 JavaScript 文件的动态导入

🔧 高度可配置

主题、行号、编辑模式等都可以自定义

📱 响应式设计

支持移动端和桌面端，自适应布局

📖 基础使用示例
简单使用：只需在 HTML 中添加 <code-editor-preview> 标签即可
⚙️ 高级配置示例
高级配置：支持多种属性配置，可以通过 JavaScript API 动态控制
🔌 JavaScript API 使用
JavaScript API：可以通过 JavaScript 动态控制组件行为
📁 外部文件导入示例
外部文件导入：可以在初始化时或运行时导入外部 JavaScript 文件，实现代码模块化
